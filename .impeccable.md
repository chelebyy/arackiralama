## Design Context

### Users
Primary audience is tourists visiting Alanya.
Secondary audience includes local customers and repeat renters.
Users want to compare vehicles quickly, understand delivery points clearly, trust the company, and move to reservation without confusion.

### Brand Personality
Corporate, trustworthy, clean.
The interface should feel professional, calm, and confidence-building rather than flashy or experimental.

### Aesthetic Direction
The public frontend should follow a corporate-minimal direction.
Default approach should be light theme only.
The design should be desktop-first, with clean spacing, strong typography, and clear visual hierarchy.
Tablet and mobile layouts must still feel solid and well-structured.
The public site must not use shadcn components, shadcn design language, dashboard aesthetics, or generic template-style visual patterns.

### Design Principles
1. Trust must be visible on the first screen: pricing, process, delivery points, and contact access should be obvious.
2. Reservation is the core journey: the main CTA and search surface should move users quickly toward vehicle selection.
3. Desktop-first decisions should drive layout, but tablet and mobile must remain clean and usable.
4. The public face must be visually separate from the admin area in layout language, components, and overall tone.
5. Simplicity should reduce friction, not remove important information.
